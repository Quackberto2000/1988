Tipos de routers

A continuacion te presentamos algunos.

Router inalambrico

Este tipo de router es el que se usa más comúnmente en las redes domésticas e incluso en algunas oficinas. Se caracteriza por estar formado por un dispositivo que se conecta a la red por medio de una conexión inalámbrica. Estos routers suelen estar equipados con una antena para poder ampliar la cobertura de la señal y así permitir que más dispositivos se puedan conectar a la red..

Read more

Router cableado

Este tipo de router es el que se usa cuando se dispone de una conexión por cable. Se caracteriza por estar formado por un dispositivo externo que se conecta al cable de la red y otro dispositivo interno que se conecta a la computadora o al modem.

Read more

Router ADSL

Es uno de los tipos de router más utilizados por las empresas y por los usuarios domésticos que disponen de una línea ADSL. Está formado por un dispositivo externo que se conecta a la línea telefónica y otro dispositivo interno que se conecta a la computadora o al modem.

Read more

Router vitual

Se trata de un tipo de router que está emulado por un software en una computadora o servidor. Esto permite a los usuarios de la red compartir el ancho de banda y la conectividad a Internet, así como también permiten a los administradores de red controlar el tráfico de red. Por eso, es muy útil cuando se necesita una Internet segura y privada.

Read more

Router movil

Se utiliza cuando para acceder a una conexión a Internet en lugares donde no hay una conexión fija disponible
